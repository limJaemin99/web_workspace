CSS 기본 속성(properties) 정리

속성명 XXX 속성값 설명
color - 색상 입력 글자 색 지정
background-color - 색상 입력 배경 색 지정
border-XXX-width top 테두리 크기 입력 윗쪽 방향의 테두리 두께를 지정
bottom 아래쪽 방향의 테두리 두께를 지정
left 왼쪽 방향의 테두리 두께를 지정
right 오른쪽 방향의 테두리 두께를 지정
border-style - 선 종류 입력 테두리 선 종류 지정
border-color - 선 색상 입력 테두리 선 색 지정
padding-XXX top 내부 여백 크기 입력 테두리 내부 윗쪽 방향의 여백 크기를 지정
bottom 테두리 내부 아래쪽 방향의 여백 크기를 지정
left 테두리 내부 왼쪽 방향의 여백 크기를 지정
right 테두리 내부 오른쪽 방향의 여백 크기를 지정
margin-XXX top 외부 여백 크기 입력 테두리 외부 윗쪽 방향의 여백 크기를 지정
bottom 테두리 외부 아래쪽 방향의 여백 크기를 지정
left 테두리 외부 왼쪽 방향의 여백 크기를 지정
right 테두리 외부 오른쪽 방향의 여백 크기를 지정

추가 설명

# 테두리 속성은 다음과 같이 한번에 작성할 수 있다.

style="border-width: 1px; border-style: solid; border-color: black;"

     ▶ border: 1px solid black
# border-width 에서 방향을 정하지 않을 경우 4방향 전부 설정한다.

# padding 에서 방향을 정하지 않을 경우 모두 동일한 크기만큼 내부 여백이 생긴다.

 방향을 생략하고 값을 입력할 경우 다음과 같이 사용할 수 있다.

① padding: 40px 30px 20px 10px; ▶ 시계방향으로 위,오른쪽,아래,왼쪽

② padding: 40px 20px; ▶ 첫번째 값은 위,아래/두번째 값은 좌,우

③ padding: 20px ▶ 모든 방향을 동일하게 지정
# margin 은 특별히 auto 를 설정하여 좌/우 여백을 중앙으로 배치되게 할 수 있다.
# width 는 너비, height 는 높이를 뜻하며, 절대값으로 px 등을 줄 수 있고, 부모의 너비 기준으로 상대값으로 %(백분율)을 줄 수 있다.

# div 는 블록 요소와 인라인 요소를 그룹으로 묶을 때 사용한다.

 span 은 인라인 요소를 그룹으로 묶을 때 사용한다.

● 인라인 요소 ⊃ 인라인 요소 ▶ O

● 인라인 요소 ⊃ 블록 요소 ▶ X

● 블록 요소 ⊃ 인라인 요소 ▶ O

● 블록 요소 ⊃ 블록 요소 ▶ O
# style 태그 안에서 값을 줄 때는 선택자 를 사용한다.

선택자에서 'td:nth-child(3)' 처럼 n번째 자식을 선택할 수 있다.
