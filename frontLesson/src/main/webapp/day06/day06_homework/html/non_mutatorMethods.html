비변 메소드

메소드 설명 Test - 콤마( , )로 구분 실행 결과
forEach( ) 배열의 각 요소에 대해 지정된 함수를 순서대로 한 번씩 호출
※ 예시에서는 각 요소들을 띄어쓰기로 구분하여 출력하는 함수 사용
filter( ) 배열의 각 요소에 대해 주어진 함수를 호출하여 그 결과가 true인 요소들로 이루어진 새로운 배열을 반환
※ 예시에서는 주어진 배열에서 짝수만 필터링
find( ) 배열의 요소를 탐색하면서 주어진 판별 함수를 만족하는 첫번째 값을 반환
※ 예시에서는 주어진 배열에서 짝수를 필터링하여 첫번째 값을 반환
findIndex( ) find( )와 동일하나, 값 대신 인덱스 숫자를 반환
※ 예시에서는 주어진 배열에서 짝수를 필터링하여 첫번째 인덱스를 반환
includes( ) 배열에 특정 값이 포함되어있는지 확인하여 true/false 반환
join( ) 배열의 모든 요소를 주어진 구분자로 합침
